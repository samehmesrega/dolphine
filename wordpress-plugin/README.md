# Dolphine - بلجن ووردبريس

يربط نماذج ووردبريس بتطبيق Dolphine بحيث يظهر كل إرسال نموذج كـ **ليد** في التطبيق تلقائياً.

## التثبيت

1. انسخ مجلد `wordpress-plugin` إلى `wp-content/plugins/` في موقع ووردبريس (يمكن تسمية المجلد `dolphine`)، أو ارفع المجلد كـ ZIP من لوحة التحكم → إضافة → رفع بلجن.
2. فعّل البلجن **Dolphine** من **الإضافات**.

## الإعداد

1. في **تطبيق Dolphine**: اذهب إلى **الربط (ووردبريس / ووكومرس)** → **نماذج ووردبريس → ليدز**.
2. اضغط **إضافة اتصال**: اختر اسماً (مثل "نموذج اتصل بنا") واختيارياً اكتب الـ shortcode للفورم (للتوثيق).
3. انسخ **رابط الويب هوك** (يبدأ بـ `https://.../api/webhooks/leads/...`).
4. في **ووردبريس**: الإعدادات → **Dolphine** → الصق الرابط → **حفظ**.

## النماذج المدعومة

- **Contact Form 7**: يعمل تلقائياً. تأكد أن الحقول تحتوي على اسم وهاتف (مثلاً `your-name`, `your-phone`, `your-email`).
- **Elementor Forms**: يعمل تلقائياً مع Elementor Pro.
- **أي نموذج آخر**: استخدم من الكود:
  ```php
  do_action('dolphine_form_submitted', [
      'your-name'  => 'اسم العميل',
      'your-phone' => '01234567890',
      'your-email' => 'email@example.com',
  ]);
  ```

## أسماء الحقول

التطبيق يقبل أسماء شائعة مثل: `name`, `phone`, `email`, `your-name`, `your-phone`, `your-email`, `address`. أي حقل يحتوي على "phone" أو يبدأ بـ "tel" يُستخدم كرقم الهاتف.

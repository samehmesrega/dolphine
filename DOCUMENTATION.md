# دولفين (Dolphin) — الوثائق الكاملة

## 1. التعريف

**دولفين** نظام ويب لإدارة الليدز والمبيعات: استيراد الليدز، إدارة التواصل، توزيعها على الموظفين، وتحويلها لعملاء وطلبات مع رفع الطلبات على ووكومرس.

---

## 2. الأقسام والشاشات

### 2.1 السايدبار

| القسم | الشاشات |
|--------|---------|
| داشبورد | أرقام، فلتر زمني، أداء السيلز، مخططات |
| ليدز | قائمة، تفاصيل، إضافة يدوي، استيراد Excel، إعدادات المصادر، ليدزي، طلبات، إدارة الحالات |
| عملاء | قائمة، تفاصيل (حسب رقم الفون) |
| طلبات | قائمة، تفاصيل |
| طلبات بانتظار الحسابات | قائمة منفصلة |
| تقارير | تقارير مفصلة |
| منتجات | مزامنة من ووكومرس |
| شيفتات | قائمة، تكوين |
| حسابات | مستخدمون، أدوار، صلاحيات |

### 2.2 الأدوار

سوبر أدمن، أدمن، مدير فريق السيلز، موظف سيلز، موظف حسابات، مسئول أوبريشنز.

### 2.3 بيانات الليد

**ثابتة:** الاسم، الموبايل، الواتساب، الإيميل، العنوان  
**مخصصة:** 5 حقول (من الإعدادات)

### 2.4 مصادر الليدز

Elementor، Contact Form 7، إدخال يدوي، Excel، جوجل شيت.

### 2.5 التواصل

أنواع: واتساب، مكالمة، تواصل فيزيائي، إيميل. مع كل تواصل: ملخص، تحديث حالة، طلب رد (اختياري).

### 2.6 الربط

رقم الفون = العميل. نفس الرقم = عميل واحد.

### 2.7 مسار الطلب

ليد → طلب مؤكد → فورم (شحن، منتجات، صورة تحويل، دفع) → إنشاء طلب → تأكيد الحسابات → رفع على ووكومرس.

---

## 3. التقنيات

- **Backend:** Node.js + Express + Prisma
- **قاعدة البيانات:** PostgreSQL
- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **المصادقة:** JWT

---

## 4. الأوامر

```bash
# Backend
cd backend && npm run dev

# Frontend
cd frontend && npm run dev

# قاعدة البيانات (عند توفر PostgreSQL)
cd backend && npx prisma db push && npm run db:seed
```
